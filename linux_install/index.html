Udisk-Polkit trick

I had a problem that each time I try to mount my external disk, I had to enter my password. To solve this problem, we can add a rule for polkit, stating that no password is needed while I'm mounting an external partition.

Add the following content into /etc/polkit-1/rules.d/10-udisks2.rules

udisk.rules

You can download it here.
